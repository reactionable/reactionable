## Install

```sh
npm install @reactionable/core
```

## Dev

### Use reactionable in local

1. In __reactionable__ dir
```bash
npm run link;
npm start
```

2. In react  app dir

```bash
npm link react react-dom @reactionable/core @reactionable/ui-bootstrap @reactionable/amplify;
npm start;
```

### Run tests

```sh
npm run test
```
