<Title>Icon</Title>

<Subtitle>Definition</Subtitle>

Icon component helps to render [Fontawesome](https://fontawesome.com/) icon. It is based on [react-fontawesome](https://github.com/FortAwesome/react-fontawesome).

<Subtitle>Usage</Subtitle>
How to render an icon (template):

```tsx
import { [Icon Name] } from "@fortawesome/free-solid-svg-icons";
import { Icon } from '@reactionable/ui-bootstrap/lib/icon/Icon';

const MyIcon = () => (
  <Icon icon={[Icon Name]} />
);
```
