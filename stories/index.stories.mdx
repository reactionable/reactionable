### ‚úçÔ∏è Manual Installation

**`@reactionable/core`** is the main package of **Reactionable**, it may be enhanced with additionnal third party packages:

- **UI**: add third party styling to user interface components.
- **Router**: integrate with third party routing (Link, navigation...).
- **Backend**: integrate with third party backend service (authentication, api...).

Each of them are requiring `@reactionable/core`, so it don't need to install the Core package if a third party package is installed

---

Install only <LinkTo kind="Core/Home">@reactionable/core</LinkTo>, or install a third party package:

| UI Packages                                                                                                    | Router Packages                                                                                                         | Backend Packages                                                                                           |
| -------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| <LinkTo kind="UI Bootstrap/Home">UI Bootstrap</LinkTo> [`React Bootstrap`](https://react-bootstrap.github.io/) | <LinkTo kind="NextJs/Home">NextJs</LinkTo> [`NextJS`](https://nextjs.org/)                                              | <LinkTo kind="Amplify/Home">Amplify</LinkTo> [`AWS Amplify`](https://docs.amplify.aws/)                    |
| <LinkTo kind="UI Material/Home">UI Material</LinkTo> [`Material-UI`](https://material-ui.com/)                 | <LinkTo kind="Router DOM/Home">Router DOM</LinkTo> [`React Router DOM`](https://reactrouter.com/web/guides/quick-start) | <LinkTo kind="GraphQL/Home">GraphQL</LinkTo> [`Apollo GraphQL`](https://www.apollographql.com/docs/react/) |
