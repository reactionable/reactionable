## Development

<br />

### Use reactionable in local

#### 1. In `reactionable` dir

```sh
npm run link; # Will print the link command to run in the react application dir
npm start;
```

#### 2. In `react application` dir

Execute the command printed by the previous `npm run link` command

### Tests

```sh
npm test;
```

### Storybook

#### 1. Execute storybook

```sh
npm run storybook;
```

#### 2. Visit [http://localhost:6006/](http://localhost:6006/)
